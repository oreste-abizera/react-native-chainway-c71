# react-native-chainway-c71

React native wrapper for the chainway sdk

## Installation

```sh
npm install react-native-chainway-c71
```

## Usage

```js
import ChainwayC71 from "react-native-chainway-c71";

// ...

const result = await ChainwayC71.multiply(3, 7);
```
